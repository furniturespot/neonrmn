Instagram API Integration Guide

How to connect 7 Different Instagram Accounts:

  1. For each category (Sports, Tech, etc.), you need a separate Instagram account (or at least separate tokens if managing multiple via Meta).
  2. Go to Meta for Developers.
  3. Generate a User Token and User ID for each specific Instagram account you want to source news from.
  4. Open this file code and locate the CONFIG.ACCOUNTS object.
  5. Paste the specific token and userId into the corresponding category slot.
  6. Set USE_MOCK_DATA: false to activate the live connection.
ACCOUNTS: {
  sports: { token: '...', userId: '...' },
  tech: { token: '...', userId: '...' },
  ...
}
